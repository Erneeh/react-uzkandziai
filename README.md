# Client Landing Page 🛍️

My first client project — a landing page built for a small business selling handmade products.
First time using React with TypeScript and Vite together.

**Built with:** React · TypeScript · Vite · Tailwind CSS

## What I built
- Responsive header with active link highlighting via `useLocation`
- Mobile navigation (centered dialog menu with custom styling)
- Shared nav links via props passed from `App.tsx` to both header and footer — no duplication
- Client-side routing with `routes.tsx`
- Active state synced across both header and footer simultaneously

## What I learned
This was my first real project with this stack. I learned how to read and adapt existing components rather than building everything from scratch, how React Router works in practice, and how to solve the data-duplication problem with props.

## Getting Started
```bash
npm install
npm run dev
```
